## Quest Participant Dashboard

### Figma Design

Figma layout: [Wallet UI](https://www.figma.com/design/0pGK686ZLE68HmI3Wu2EvF/Survexa?node-id=647-2951&t=7mOUCjB6dSAYh8Xb-1)

### Responsive Screens

The dashboard includes desktop and tablet responsive layouts.

Included views:

* Desktop Dashboard
* Tablet Dashboard
* Quest Discovery Tabs
* Recent Earners Panel
* Recently Submitted Panel

### Asset Exports

Optimized assets are available in the project assets folder.

Formats used:

* `.svg` for icons and interface graphics
* `.webp` for optimized image/avatar assets

### Reusable Components

The following reusable dashboard components were created:

* Dashboard Welcome Banner
* Quest Discovery Tabs
* Quest Card
* Recent Earner Item
* Recent Submission Item
* Wallet Balance Indicator
* Notification Button
* Profile Menu
* Activity Panel
* Dashboard Layout Container

### Notes

The dashboard follows the platform’s dark-themed visual identity with clear hierarchy, readable typography, consistent spacing, and strong emphasis on quest rewards, engagement activity, and earning opportunities.
